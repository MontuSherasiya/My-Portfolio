## 🚀 Features

- ⚡ **Zero Dependencies** — Pure HTML, CSS & Vanilla JS, no frameworks needed
- 🌑 **Dark Premium Theme** — Deep dark background with electric cyan/teal accents
- 🖱️ **Custom Cursor** — Animated cursor with smooth trailing ring effect
- 💻 **Terminal Card** — JSON-styled developer card in the hero section
- 📊 **Animated Skill Bars** — Progress bars that animate on scroll into view
- 🎞️ **Scroll Animations** — Staggered reveal animations for all sections
- 📱 **Fully Responsive** — Looks great on mobile, tablet, and desktop
- 🔗 **All Links Live** — GitHub, LinkedIn, email, and phone all connected
